# Sandstone Stake Pool

Website for Sandstone, a Cardano stake pool based in Perth, Western Australia.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Environment Variables

Create a `.env.local` file with:

```
CARDANOSCAN_API_KEY=your_api_key_here
```

## Deployment

Deployed on Netlify. Environment variables must be configured in the Netlify dashboard.
